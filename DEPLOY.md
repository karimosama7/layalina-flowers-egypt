# تعليمات النشر (Deploy)

## الطريقة الأسهل - Netlify Drop

1. افتح https://app.netlify.com/drop
2. اسحب مجلد `dist` إلى الصفحة
3. سيتم إعطاؤك رابط مباشر!

## طريقة أخرى - Vercel

1. افتح https://vercel.com
2. سجل الدخول بحساب GitHub
3. اضغط "New Project"
4. اسحب مجلد المشروع أو اربطه بـ Git
5. Vercel سيكتشف Vite تلقائياً

## طريقة أخرى - GitHub Pages

1. ارفع المشروع على GitHub
2. في Settings > Pages
3. اختر Source: GitHub Actions
4. سيتم النشر تلقائياً
